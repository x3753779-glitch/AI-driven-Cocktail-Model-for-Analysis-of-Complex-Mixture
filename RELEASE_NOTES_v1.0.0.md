# v1.0.0 – Initial public release

This is the initial public release of Cocktail Model: an online platform for database expansion and model application.

## Included in this release

- Source code for the reproducible analysis workflow.
- Data-processing workflow and configuration files for regenerating manuscript-related outputs.
- Trained model artifacts and associated model metadata.
- Example data and processed data used by the reproduction package.
- Generated figures, tables, metrics, and run logs.
- Three demonstration videos:
  - `videos/cocktail-model_detection_experiment.mp4`
  - `videos/cocktail-model_application.mp4`
  - `videos/online_liquid-analysis_interface_workflow.mp4`
- Hugging Face online liquid-analysis interface:
  - https://huggingface.co/spaces/SXHIR/liquid-analyzer
