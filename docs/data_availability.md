# Data Availability

The data and artifacts supporting the released results are available in this repository.

This release package includes:

- Source code for the reproduction workflow.
- Packaged example/source data and processed data.
- Trained model artifacts and model metadata.
- Reproduced figures, tables, metrics, and run logs.
- Three supplementary demonstration videos.
- A linked Hugging Face online demonstration interface.

The online demonstration interface is available at:

https://huggingface.co/spaces/SXHIR/liquid-analyzer

Supplementary videos:

1. `videos/cocktail-model_detection_experiment.mp4`: cocktail-model detection experiment.
2. `videos/cocktail-model_application.mp4`: cocktail-model application workflow.
3. `videos/online_liquid-analysis_interface_workflow.mp4`: online liquid-analysis interface workflow.

For manuscript citation, the GitHub Release `v1.0.0` should be archived through Zenodo, and the resulting Zenodo DOI should be cited as the software/data release DOI. The Zenodo Creator for this software/data release should be Xiaohe Shang only.
