# Data availability

The data supporting the findings of this study are available in the paper and in this repository.

This repository contains the processed data, packaged model artifacts, generated figures, generated tables, quantitative metrics, and supplementary demonstration videos used for reproduction of the released results.

The online demonstration interface is available at:

https://huggingface.co/spaces/SXHIR/liquid-analyzer

Supplementary videos:

1. `videos/Supplementary_Movie_S1_cocktail_demo.mp4`: cocktail-mixture demonstration video.
2. `videos/Supplementary_Movie_S2_blood_demo.mp4`: blood-mixture demonstration video.
3. `videos/Supplementary_Movie_S3_online_website_demo.mp4`: online website demonstration video.
