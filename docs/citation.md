# Citation

If you use this repository, cite the archived software release and the associated manuscript.

The Hugging Face online site, GitHub repository, source code, trained models, data-processing workflow, and demonstration contents were independently created by Xiaohe Shang. The software repository and website citation author should therefore be `Shang, X.` only. The full manuscript author list should be used only for the paper itself and paper metadata.

## Recommended software citation

```text
Shang, X. (2026). Cocktail Model: an online platform for database expansion and model application (v1.0.0) [Software]. Zenodo. https://doi.org/10.5281/zenodo.21470719
```

## Zenodo archival

The GitHub Release `v1.0.0` has been archived with Zenodo. The version DOI is:

```text
10.5281/zenodo.21470719
```

The Zenodo Creator should be Xiaohe Shang only.

## Manuscript citation

After the manuscript is accepted or published, cite the final paper in addition to the software release.

## Metadata

Machine-readable citation metadata are provided in the root-level `CITATION.cff` file. GitHub and Zenodo can use this file to generate a recommended citation.
