# Results Directory

Running `python reproduce.py` rewrites this directory, except this README.

Expected outputs:

- `figures/`: Reviewer-facing figure PNG files.
- `tables/`: Reviewer-facing CSV/XLSX tables.
- `metrics.json`: Aggregated quantitative results.
- `metrics_summary.csv`: Compact metric table under `tables/`.
- `run_log.txt`: Step-by-step run log.
- `run_summary.txt`: Generated output summary.
